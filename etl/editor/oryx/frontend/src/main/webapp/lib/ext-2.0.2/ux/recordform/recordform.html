Description

This plugin scans grid's store and column model and creates a form, initially hidden, that is suitable for record editing.

The form has simple column layout with configurable number of columns and displays fields with values initially loaded from the underlying record.

The plugin does not support any user defined layouts or more complex structures, besides built-in column layout, such as fieldsets or other arrangement of fields.

If a column model editor is available for a field this editor is used, if not the suitable editor is guessed from the underlying record field type.

Fields displayed in the form can be selectively ignored and fields that to not have their corresponding columns in the grid can be edited too.

The form by itself does not send any values to the server, it updates the underlying record instead.

The plugin plays best with RowActions Plugin, as it is in this example, but it is also possible to open the form from row dblclick event.
